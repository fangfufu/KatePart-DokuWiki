# KatePart-DokuWiki
A KatePart syntax highlighting file for DokuWiki 

## Instruction
    Instruction: Copy this file to ~/.kde/share/apps/katepart/syntax
    Your DokuWiki text file also needs the extension of ".dokuwiki".
